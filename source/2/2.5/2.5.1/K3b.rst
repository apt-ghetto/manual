Chapter 2.5.1 K3b
=================

K3b is the default CD or DVD buring software on Lubuntu. 

Version
-------
Lubuntu ships with version 18.04.3 of K3b. 

Screenshot
----------
.. k3b.png

How to Launch
-------------
To launch K3b from the menu -> sound and video -> K3b or run 

.. code:: 

   K3b 
   
from the command line.
